⏺ Текущее состояние проекта virt_card_processing (YeezyPay)

  Проект — платформа для выпуска и обслуживания виртуальных карт с криптодепозитами. Стек: FastAPI + Tortoise ORM (async backend), Django (admin), React + TypeScript (Telegram Mini App),
   PostgreSQL + Redis + NATS JetStream (инфра), Docker Compose (11 сервисов).
